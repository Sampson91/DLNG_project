你是如何合成数据集的？，为什么要分训练集，验证集和测试集？

回答：

问题 3¶

你最终的模型架构是什么样的？（什么类型的模型，层数，大小, 如何连接等）

回答：

该模型通过卷积神经网络采用3×3像素的尺寸的窗口在图像中进行滑动，对于每个尺寸方法采用纹理、颜色或亮度对邻近的像素进行聚合，从而对物体进行识别。由于卷积层获取的图片尺寸没有发生变化，而为了减少模型的复杂度，通过压缩图片来实现即通过神经网络中下采样来稀疏参数，该模型采用最大值池化Max_pool方法来实现;对于网络中每一层的卷积结果需要通过激活函数对数据进行分割，该模型通过relu激活函数对数据分割，由于relu函数1.单侧抑制 2相对宽阔的兴奋边界3.稀疏激活性，所以惨用relu函数为激活函数。模型中为了防止模型过拟合，通过dropout函数随机选择一定比例的某些神经元的输出结果数据进行丢弃，从而使得模型减少对特定的神经元连接片段太过敏感，也就迫使网络去学习更加鲁棒的特征。

模型采用keras搭建的10层卷积升神经网络，第一层为input输入层对（28,140,1）的图像进行输入，（2,4,6）层为（32,64,128）核尺寸为3×3的卷积层，（3,5,7）为2×2的池化层，第八层为展开层，把数据转换为一维，第九层为dropout层（即丢弃某些神经元的输出结果），第十层为5个结果的输出层。

def evaluate(model,X_test, y_test):
    # TODO: 按照错一个就算错的规则计算准确率
    y_test_predict = model.predict(X_test)
    y_test_predict = np.array(y_test_predict)
    y_test_predict = np.argmax(y_test_predict, axis = 2)
    y_test = np.argmax(y_test,axis = 2)
    y_test_predict = y_test_predict.T
    y_test = y_test.T
    number = 0
    for i in range(len(y_test_predict)):
        if (y_test_predict[i] == y_test[i]).all():
            number += 1
